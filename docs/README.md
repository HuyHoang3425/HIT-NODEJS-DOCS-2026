# 🚀 LỘ TRÌNH KHÓA HỌC NODE.JS 2026

|   Buổi    | Nội dung trọng tâm          | Chi tiết bài học                                                                                                                                                   |
| :-------: | :-------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|   **1**   | **Tổng quan & Ôn tập JS**   | Giới thiệu Node.js, cài đặt. Ôn tập JavaScript (ES6+, Async/Await, Promise).                                                                                       |
|   **2**   | **Express.js cơ bản**       | Express.js là gì? Thiết lập dự án đầu tiên. HTTP Methods (GET, POST, PUT, DELETE).                                                                                 |
|   **3**   | **Template Engine & MVC**   | Làm việc với EJS. Tìm hiểu mô hình Model - View - Controller trong Express.                                                                                        |
|   **4**   | **CRUD API với JSON**       | Đọc/ghi file bằng module `fs`. Xây dựng API thao tác dữ liệu qua file JSON.                                                                                        |
|   **5**   | **MongoDB & Mongoose**      | NoSQL là gì? Cài đặt MongoDB Atlas/Compass. Kết nối Server với Mongoose.                                                                                           |
|   **6**   | **Middleware & Validation** | Xử lý lỗi (Error Handling), Validation dữ liệu đầu vào với thư viện Joi.                                                                                           |
|   **7**   | **Authentication & JWT**    | Phân biệt Authentication & Authorization. Cơ chế xác thực với JSON Web Token.                                                                                      |
|   **8**   | **Authorization**           | Phân quyền RBAC (Admin/User) và Middleware bảo vệ Router.                                                                                                          |
|   **9**   | **Upload File**             | Sử dụng thư viện Multer để upload và quản lý file cục bộ hoặc Cloud.                                                                                               |
|  **10**   | **Quan hệ dữ liệu**         | One-to-Many, Many-to-One. Truy vấn nâng cao: Lọc, Tìm kiếm, Phân trang.                                                                                            |
| **11,12** | **Củng cố kiến thức**       | Ôn tập các phần chưa vững, giải đáp thắc mắc, Hướng dẫn đưa ứng dụng lên Server (Render, Vercel). Giới thiệu Nodemailer, Socket.IO (Real-time), Caching với Redis. |
